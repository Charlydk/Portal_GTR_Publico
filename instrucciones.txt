
para inciar en casa:

BACKEND
Navega a la carpeta backend de tu proyecto:


cd RUTA/A/TU/PROYECTO/PORTAL_GTR/backend
Activa el entorno virtual. Este paso es muy importante:

en caso de no tener el venv instalado
python -m venv venv

en caso que la politica este desactivada usar:
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser

.\venv\Scripts\activate
(Deberías ver (venv) al principio de la línea de la terminal).

para instalar todas las librerias
pip install -r requirements.txt

Inicia el servidor con Uvicorn:

uvicorn main:app --reload --port 8001

USa este link para swagger
http://127.0.0.1:8001
--------------------------------------------
FRONTEND
**En casa**
Navega a la carpeta del frontend:

cd RUTA/A/TU/PROYECTO/PORTAL_GTR/frontend/portal-gtr-frontend
Inicia el servidor de desarrollo de Vite:

npm run dev
